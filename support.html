Common Issues

Problem Fix
App shows "Setup Required" You need to paste your Google Apps Script URL in the app settings screen.
Login not working Check that your Google Apps Script is deployed and the URL is correct. Make sure access is set to "Anyone".
Data not loading Pull down to refresh. If it persists, check your internet connection and re-deploy your Apps Script.
Changes not saving After editing your Apps Script, always create a new deployment for changes to take effect.
Inventory numbers look wrong Run the runSetup function in Apps Script to reset sheet structure, then re-enter data.

Getting Started

  • Create a Google Sheet and add the Apps Script backend
  • Deploy the script as a Web App with access set to "Anyone"
  • Copy the Web App URL and paste it into the app's setup screen
  • Run runSetup to create the required sheet tabs
  • Log in with your admin credentials
